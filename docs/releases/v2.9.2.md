# GitBook Worker v2.9.2 Release Notes

**Release date:** May 12, 2026
**Focus:** landscape table context packets, compact reference/source list handling, and customer-observed PDF rest-page cleanup.

## Summary

Version 2.9.2 is a focused layout hotfix after the v2.9.1 customer acceptance
hardening. It improves the table preprocessing strategy so landscape tables keep
short editorial context with them when the packet is bounded and likely to fit.

The release is intentionally generic: customer-observed layout issues are
covered through anonymized regression fixtures. No customer raw content, PDFs,
extracted text, rendered page images, logs, or diagnostic dumps are included in
the repository or release source tree.

## Fixed

- Kept short headings, introductory paragraphs, blockquote notes, and compact
  legends with wrapped landscape tables.
- Kept separator plus compact `Querverweise` / `References` blocks with the
  preceding landscape table when they are a short trailing reference section.
- Kept compact `Quellen & Verweise` / `Sources & References` numbered or
  bulleted lists with the preceding landscape table when the height check allows
  it.
- Collapsed adjacent geometry wrappers so separator-only pages are not left
  between consecutive landscape packets.
- Converted Markdown `**strong**` spans inside generated LaTeX longtable cells to
  `\textbf{...}` instead of leaving literal asterisks in the PDF.

## Added

- Height-aware landscape packet estimation. Landscape table packets can promote
  from A4 landscape to larger landscape paper when context plus table content
  would otherwise be too tall.
- Anonymized regression tests for:
  - short intro and heading context,
  - compact legend tables,
  - chapter lead packets,
  - one short sibling table,
  - compact trailing references,
  - compact trailing source/reference lists,
  - adjacent wrapped packet cleanup,
  - height-based landscape paper promotion.
- Concept documentation for context packets and source/reference list handling.

## Known Limitations

- The packet strategy is conservative. It intentionally does not pull arbitrary
  next sections into landscape pages; only bounded table context and recognized
  compact reference/source sections travel with a wrapped table.
- Page numbers can shift after layout cleanup because isolated rest pages are
  removed.
- Customer acceptance may still report editorial warnings that require content
  decisions; v2.9.2 addresses table layout packet behavior, not content policy.
- No Noto fonts are used or recommended. Apart from DejaVu, project fonts and
  emoji expansion remain on the CC BY 4.0 ERDA/Twemoji path.
