# Fehlerhafte Referenzen

Diese Beispiele sind absichtlich unsauber. Der Checker soll sie nicht blind
ueberschreiben, sondern im Report nachvollziehbare Korrekturvorschlaege liefern.

## Quellen

1. W3C WCAG 2.2, irgendwann, Link kaputt: https://www.w3.org/TR/WCAG999/
2. Unicode Standart 15, falscher Titel und Tippfehler, ohne Herausgeber.
3. Python docs, Version unbekannt, http://docs.python.org/does-not-exist
4. "Pandok Manual" ohne Autor, falscher Name, https://pandoc.org/not-the-manual
5. Creative Common License 4, unvollstaendige Lizenzangabe ohne URL.
6. DOI 10.0000/not-a-real-doi mit frei erfundenem Titel.
