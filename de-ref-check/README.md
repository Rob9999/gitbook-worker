# AI Reference Check Testbaum

Dieser Sprachbaum ist ein isolierter QA-Fundus fuer den optionalen
`ai-reference-check`. Er enthaelt gute, fehlerhafte und exotische Quellenformen,
damit Parser, LLM-Prompting, Report-Generierung und menschliche Review-Flows
getrennt von den produktiven Buchbaeumen getestet werden koennen.

Der Baum ist nicht fuer regulaere PDF-Builds aktiviert.
