# Chapters

This folder contains the main chapters of the ERDA Book with structured content, examples, and demonstration material.

## Available Chapters

- **[Chapter 1 – Observable Patterns](chapter-01.md)**: Introduction to methodical observations and structured data collection. Demonstrates neutral descriptions of measurement procedures and documentation techniques.
  
- **[Chapter 2 – Tables and Data Structures](chapter-02.md)**: Overview of table layouts, formatting, and presentation forms for structured data.

## Chapter Structure

Each chapter follows a consistent structure:
- **Introduction**: Context and objectives
- **Methodical Steps**: Systematic approach
- **Examples**: Concrete use cases with neutral datasets
- **Cross-references**: Links to appendices and supplementary materials

## Usage

The chapters serve as:
- Test material for PDF generation and layout validation
- Reference for Markdown-to-PDF conversion
- Demonstration content for multilingual documentation

---

*All chapters use neutral data without personally identifiable information.*
